# CrossConnect v2 🎮

> Retro arcade pixel quiz battle across Indian states. P2P · Free · No server.

## What's New in v2

- 🎮 **Retro arcade theme** — pixel art, scanlines, CRT glow, chiptune SFX
- 🧑 **Avatar builder** — choose your sprite & colour
- 🗳️ **Category voting** — both players vote; ties broken by host
- ⚡ **Power-ups** — Skip (skip a question), Freeze (lock opponent for 5s), 2X (double points)
- ⭐ **Bonus questions** — Q5 and Q10 are always worth 2x points
- 🏆 **Score system** — points-based, not just speed
- 💀 **Arcade result screen** — VICTORY or GAME OVER

## How to Play

1. Fill in name, state, city → Build your avatar
2. Create or join a room with a 6-letter code
3. Vote for a category (Food / Culture / Sports / Mixed)
4. Answer 10 questions — use power-ups strategically
5. Higher score wins the key 🗝️ and unlocks chat
